# 🌿 Planty — Site vitrine WordPress personnalisé Projet réalisé dans le cadre du parcours **Développeur WordPress** chez [OpenClassrooms](https://openclassrooms.com/). --- ## 🎯 Objectif Développer un site vitrine WordPress pour **Planty**, une marque de boissons énergisantes à base de plantes. ### Objectifs fonctionnels : * Créer un site responsive et fluide avec 4 pages (Accueil, Contact, Commande, Mentions légales) * Rendre tous les contenus modifiables sans coder * Intégrer un formulaire fonctionnel et un lien vers le back-office conditionnel (admin connecté) * Respecter les bonnes pratiques (W3C, accessibilité, performances) --- ## 🧩 Ressources et outils fournis * ✅ Maquette Figma + Prototype * ✅ Spécifications techniques complètes * ✅ Contenus (textes/images) --- ## 🛠️ Stack technique * WordPress + Gutenberg (éditeur de blocs) * Thème [ASTRA](https://wpastra.com/) personnalisé * HTML / CSS / PHP * Plugins utilisés : * 🔌 **Spectra** : blocs avancés pour Gutenberg * 🔌 **Fluent Forms** : gestion des formulaires * 🔌 **All-in-One WP Migration** : sauvegarde + base de données * 🔐 Extensions de sécurité & mise en cache --- ## ⚙️ Développements spécifiques | Élément | Description | | -------------- | ------------------------------------------------------------------------------- | | 📬 Formulaires | 2 formulaires (contact & commande) fonctionnels via Fluent Forms | | 📤 Email | Envoi vers : `planty.drinks@gmail.com` | | 🔗 Lien Admin | Ajout via hook `wp_nav_menu_header-menu_items` (visible uniquement si connecté) | | 💡 Shortcode | Création d’un shortcode pour bordure arrondie personnalisée | | 🎨 Responsive | Site totalement responsive à partir de 320px | | 🧹 HTML/CSS | Code |
